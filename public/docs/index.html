    User management

    APIs for managing users

    Create a new user

    Make an empty POST request to register a new user. The endpoint returns an api_token that you should store on the client. The api_token cannot be displayed in plain text again.

    Example request:

    curl -X POST "http://localhost/api/users" 
    const url = new URL("http://localhost/api/users");

let headers = {
    "Accept": "application/json",
    "Content-Type": "application/json",
}

fetch(url, {
    method: "POST",
    headers: headers,
})
    .then(response => response.json())
    .then(json => console.log(json));

    Example response (200):

    {
    "meta": {
        "api_token": "q6gi1gCeDA7nc4gTHs1LUuBPTgjuYXy8X8Zze33QFxCAGthQ7ebhApluudhr"
    },
    "data": {
        "id": "b395592d-26f1-4da4-af8b-51ea294d0128",
        "created_at": "2019-03-11T21:21:26+00:00"
    }
}
